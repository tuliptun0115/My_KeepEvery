# 靈感收藏盒 (InspirationBox)

一款輕量、優雅的靈感採集工具。透過 LINE 傳送連結，AI 自動摘要標籤，並歸檔至 Google Sheets。

## 🚀 快速開始
1. `npm install`
2. 配置 `.env.local`
3. `npm run dev`

## 🛠️ 技術棧
- **Frontend**: Next.js 15, Tailwind CSS, shadcn/ui.
- **Backend**: API Routes.
- **AI**: Gemini 1.5 Flash.
- **Integration**: LINE Message API, Google Docs API.
