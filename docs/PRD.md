# PRD - 靈感收藏盒

## 💎 產品願景
打造最溫暖且無感的「靈感捕捉」空間，讓使用者在 LINE 中隨手收藏美好，並自動分類與歸檔。

## 核心功能
### 1. LINE 萬物採集
- 使用者傳送 URL 或文字，系統紀錄。
- 直到接收「儲存到google Word」關鍵字，才觸發 AI 與寫入流程。
- **限制**：URL 僅抓取 `<title>`，不爬取全文。

### 2. AI 內容強化
- 提取標題後，Gemini 生成 3 個帶有 `#` 的標籤。
- 格式統一：`#分類1, #分類2, #分類3`。

### 3. Google Sheets 持久化
- 格式：`[時間] | [標題] | [標籤] | [來源 URL/文字]`。
- 新增內容一律 Append 在試算表最後一行。

## 視覺需求
- **色調**：櫻花粉 (#FFB7C5) + 白 + 鵝黃 (#FFF9C4)。
- **風格**：Bento Grid (便當盒網格)。
- **動態**：輕盈、微透感、淡入。
