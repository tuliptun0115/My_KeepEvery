# UI/UX 指南 - 靈感收藏盒

## 🎨 設計人格與風格
- **名稱**：插畫壓邊貼紙風 (Illustrative Sticker Style)
- **氛圍**：溫暖、手感、童趣、輕盈。
- **關鍵要素**：
  - **邊框 (Border)**：使用明顯的黑色或深色壓邊 (2px - 3px)，營造貼紙感。
  - **陰影 (Shadow)**：使用硬陰影或平移陰影 (Drop Shadow)，呈現紙張疊層感。
  - **形狀**：圓角矩形 (Rounded XL)，邊緣不完全規則的微小擾動 (可選)。

## 🌈 配色方案 (Color Palette)
- **主色 (Primary)**：櫻花粉 `#FFB7C5`
- **副色 (Secondary)**：薄荷綠 `#B2DFDB` (參考附圖基座)
- **點綴 (Accent)**：淡鵝黃 `#FFF9C4` (燈光與花蕊)
- **背景 (Background)**：極淺粉 `#FFF0F3` 到白色的漸層。

## 🍱 佈局規範 (Layout)
- **Bento Grid**：不同大小的卡片排列。
- **元素裝飾**：
  - 漂浮的雲朵 SVG。
  - 邊角點綴的手寫感花朵。
  - 模擬紙張質感的細微紋理背景。

## ✨ 動態與交互 (Interaction)
- **Framer Motion**：
  - 卡片進入時帶有「彈跳 (Bounce)」效果。
  - 懸停時卡片略微放大並增加陰影深度。
  - 櫻花瓣或雲朵的緩慢位移動態。
