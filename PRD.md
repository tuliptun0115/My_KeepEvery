# PRD - 靈感收藏庫

## 1. 產品定位

- 產品名稱：`靈感收藏庫`
- 產品類型：個人使用的收藏、回找與再利用工具，不是團隊協作平台。
- 核心問題：內容雖然收得到，但資訊太空、難以回找、後續幾乎用不起來。
- 產品方向：先讓每筆收藏或指令都能被快速理解、快速找到、快速再使用。

## 2. 主要使用情境

### 靈感收藏

- 使用者最常貼的是社群 URL，而且通常只貼網址，不會補充說明。
- 使用者也會直接把一段文字貼到 LINE。
- 除了 AI 相關內容，也會收藏一般工具、文章、觀點與其他非 AI 資訊。

### 指令收藏

- 使用者會在同一個 LINE Bot 直接貼一段 prompt / 指令。
- 使用者也會在網站端手動新增、修改與複製 prompt。
- prompt 的重點不是摘要，而是保留可直接重用的完整指令內容。

### 主要回找路徑

使用者回找靈感資料時，常用線索順序為：

1. 主題
2. 用途
3. 來源
4. 模糊印象

使用者回找 prompt 時，常用線索則更偏向：

1. 分類
2. 關鍵詞
3. 是否能快速複製再用

## 3. 正式產品能力

### A. 靈感收藏庫

- 收藏純文字、社群 URL、一般 URL。
- 收藏進來後，不再只有原始標題或空洞標籤，而是至少能快速看懂內容大意。
- 社群 URL 即使抓不到完整正文，也能低信心收錄，不直接丟棄。
- 首頁、列表頁、詳細頁皆以摘要、用途、標籤與來源支撐搜尋與回找。

### B. 指令寶庫

- 指令寶庫是獨立於靈感收藏庫的第二資料模組。
- 專門收納可重複使用的 prompt / 指令。
- 與靈感收藏庫共用同一份 Google Sheets，但使用另一張獨立 sheet。
- 支援從 LINE 自動分流建立 prompt，也支援網站端手動新增。
- 每筆 prompt 皆可修改與一鍵複製。

## 4. 靈感收藏 MVP 欄位

每筆靈感收藏至少包含以下欄位：

- `id`
- `input_type`
- `raw_input`
- `source_title`
- `source_url`
- `created_at`
- `source_platform`
- `content_type`
- `summary`
- `key_points`
- `tags`
- `use_case`
- `topic_category`
- `confidence_level`
- `parse_status`

## 5. 指令寶庫 MVP 欄位

### 使用者主要欄位

- `prompt_category`
- `prompt_text`

### 系統欄位

- `id`
- `created_at`
- `updated_at`
- `source_type`

## 6. 輸入規則

### 靈感收藏輸入

- 純文字：直接進行 AI 整理。
- 社群 URL：優先做可降級內容解析。
- 一般 URL：補抓正文、描述或片段後再整理。

### 指令寶庫輸入

- 若 LINE 訊息中含有 URL，該訊息一定不是 prompt，直接走既有靈感收藏流程。
- 若訊息不含 URL 且帶有 `/prompt` 前綴，100% 視為指令寶庫資料。
- 若訊息不含 URL 且沒有 `/prompt` 前綴，由 AI 判斷是否為 prompt。
- 若 AI 判定不是 prompt，回到既有純文字靈感收藏流程。

## 7. 分類規則

### 靈感收藏

- 靈感收藏維持既有 `use_case`、`topic_category`、`tags` 等結構化分類能力。

### 指令寶庫

- prompt 類別總數固定不超過 5 種。
- 類別由 AI 自動判斷，但只能從固定集合中擇一，不可自行創造新類別。
- 第一版固定類別如下：
  - `寫作生成`
  - `行銷文案`
  - `工作效率`
  - `開發技術`
  - `其他`

## 8. 頁面需求

### 首頁 (`/`)

- 顯示最新靈感收藏與第一層回找入口。

### 列表頁 (`/list`)

- 提供靈感收藏的搜尋、多重過濾與分頁。

### 詳細頁 (`/detail`)

- 顯示單筆靈感的完整內容與上下文導覽。

### 指令寶庫首頁 (`/prompts`)

- 輕量化展示最新 5 筆指令，不含搜尋與篩選。
- 提供「看更多指令 →」連結跳轉至完整列表頁。

### 指令寶庫列表頁 (`/prompts/list`)

- 提供完整搜尋、分類篩選與表格展示。
- 支援一鍵複製、直接編輯、從網站手動新增與進入詳情頁。

### 指令寶庫詳情頁 (`/prompts/detail`)

- 顯示完整 prompt 內容。
- 顯示分類、來源、建立時間與更新時間。
- 支援編輯與複製。

## 9. 成功標準

### 靈感收藏庫

- 收藏進來後，不再只有原始標題或空洞標籤，而是至少能快速看懂內容大意。
- 社群 URL 即使抓不到完整正文，也能低信心收錄，不直接丟棄。
- 前端能以摘要、用途、標籤與來源支撐搜尋與回找。

### 指令寶庫

- 使用者可從側邊欄進入 `指令寶庫`。
- 從 LINE 傳送 prompt 後，可正確建立一筆 prompt。
- 含 URL 的 LINE 訊息不會誤進指令寶庫。
- 使用者可從網站新增、編輯、查看與複製 prompt。
- prompt 與靈感收藏資料互不污染，各自維持清楚的資料邊界。

## 10. 本階段不追求

- 團隊協作與權限系統
- 語意搜尋、RAG、向量資料庫
- 自動推薦、每週回顧、主題聚合
- 立即替換 Google Sheets 為正式資料庫
- Prompt 批次管理、批次刪除、匯入匯出
